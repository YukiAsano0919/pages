マイ検索エンジン




方法

１）linuxにdockerを入れて$ docker pull codelibs/fess:latest とする。
２）コンテナを起動する。$ docker run -d -p 8080:8080 --name fess codelibs/fess:latest
３）ブラウザで127.0.0.1:8080にアクセスして見れるのを確認する。
４）グローバルアドレスを調べてDDNSに登録する。
５）LANでIPマスカレードの登録をする。

結果

初めてだったので大変でした。
